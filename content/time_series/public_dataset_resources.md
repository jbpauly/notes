## A collection of resources for public time series datasets

1. [Time Series Classification]('http://www.timeseriesclassification.com/dataset.php')
    - Don't be fooled by the name. These datasets can be used to practice forecasting.
2. [R datasets]('http://www.forecastingbook.com/resources/data-and-code')
3. [UCI Machine Learning Repository]('https://archive.ics.uci.edu/ml/datasets.php?format=&task=&att=&area=&numAtt=10to100&numIns=&type=ts&sort=nameUp&view=table')
4. [Kaggle]('https://www.kaggle.com/datasets?search=time+series')
5. [Data World]('https://data.world/datasets/time-series')
6. [TensorFlow (code and data)]('https://www.tensorflow.org/tutorials/structured_data/time_series')
